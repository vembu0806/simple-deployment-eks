# Simple Application Hosted on AWS EKS

This repository provides Terraform scripts to set up an AWS EKS cluster and deploy a sample Nginx application. 

## Features
- AWS EKS cluster creation with Terraform
- VPC configuration
- Nginx deployment in the cluster using Kubernetes manifest

## Prerequisites
1. **Tools Required**:
   - Terraform (>= 1.0)
   - AWS CLI
   - kubectl

2. **Permissions**:
   - IAM user with permissions to create EKS clusters and associated resources.

3. **Configurations**:
   - AWS credentials configured locally.
